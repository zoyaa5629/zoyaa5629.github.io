Movie Webpage Project

Context

In this project we learned to make a webpage about a movie that we made up. I made a movie called Survival Love with my partner Leo. The movie is about a boy and girl who meet each other in a forest and try to survive together but come across different challenges that make it harder to survive and in the end they fall in love with each other. In this project we used both HTML and CSS to talk about our movie and customize the website.

Content

In this project we used many elements such as:
  • Headings
  • Background
  • Margin
  • Links
  • A sequel

Challenges

Me and my partner had challenges while making the webpage such as the background and getting it to a gradient color as welll as adding the link to the sequel. The code was confusing and it was something had hadn't really touched upon in class but luckily me and my partner successfully did it with research online. It took many tries and days to get our answer and find out how to add a sequel link especially. We had never added a link like that and it was really confusing but once we added it to our webpage we understood how to do it now for future projects.

Takeaways

I used new code at that time for adding the sequel to the webpage which looked like this: < a href="sequel.html" > < /a >
